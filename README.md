# intrepion-to-do-json-rpc-server-csharp-dotnet-webapi

## Commands

### Build

```bash
dotnet build
```

### Test

```bash
dotnet test
```

### Run

```bash
CLIENT_URL="http://localhost:3000" dotnet run --project ToDoWebApi
```
